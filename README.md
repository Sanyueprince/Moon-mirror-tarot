# 月镜言 · 塔罗占卜 H5 Demo

一个可在手机浏览器直接运行的塔罗占卜可玩 Demo（方案 B），零安装、零注册。
深色神秘视觉风格，包含完整占卜流程：**提问 → 选牌阵 → 洗牌 → 抽牌 → 翻牌揭晓 → 解读（概要 + 深层）**。

## 📁 文件结构

```
月镜言-小程序Demo/
├── index.html      页面结构（启动/登录/首页/占卜全流程/记录）
├── style.css       深色主题样式（#0d0b1a 底 / 金 / 紫）
├── tarot-data.js   塔罗牌库（20 张含 SVG 牌面）+ 5 种牌阵 + 文案
└── app.js          交互逻辑（洗牌动效、扇形抽牌、翻牌、解读、本地记录）
```

## ✨ 功能亮点

- **真实塔罗牌面**：每张牌用 SVG 手绘，支持正/逆位（逆位自动旋转 180°）
- **5 种牌阵**：每日一牌 / 时间之流 / 抉择之路 / 恋人之心 / 凯尔特十字
- **洗牌交互**：点击牌堆洗牌，手机端支持「摇一摇」+ 振动反馈
- **扇形抽牌**：凭直觉点选，选满自动进入揭晓
- **翻牌动效**：3D 翻转揭晓每张牌
- **解读分层**：概要解读（免费）+ 深层解读（逐牌位 / 能量关系 / 行动建议 / 幸运提示，本 Demo 免费解锁）
- **每日一牌**：首页按日期固定，每天一张
- **历史记录**：占卜结果保存在手机本地（localStorage）

## 📱 如何在手机上测试

### 方法 1：本地服务器 + 同一 WiFi（推荐）

1. 电脑终端进入项目目录并启动服务：
   ```bash
   cd /Users/joanwu/Desktop/月镜言-小程序Demo
   python3 -m http.server 8080
   ```
2. 查看电脑局域网 IP（Mac）：
   ```bash
   ipconfig getifaddr en0
   ```
   假设得到 `192.168.1.10`
3. 确保**手机和电脑连同一个 WiFi**，用手机浏览器打开：
   ```
   http://192.168.1.10:8080
   ```

> 提示：「摇一摇洗牌」在 iOS Safari 需 HTTPS 或用户授权动作权限；若不生效，直接「点击牌堆」即可洗牌。

### 方法 2：直接把文件发到手机

把整个 `月镜言-小程序Demo` 文件夹通过 AirDrop / 微信 / 网盘发到手机，用支持本地网页的浏览器（如 iOS 的「Documents」App）打开 `index.html`。

### 方法 3：电脑先预览

直接双击 `index.html`，或终端运行：
```bash
open /Users/joanwu/Desktop/月镜言-小程序Demo/index.html
```
建议用浏览器开发者工具的「移动设备模拟」查看（推荐 iPhone 尺寸）。

## 🔮 体验路径

启动页 → 登录（可点「先随便看看」）→ 首页「开始占卜」→ 选分类/输入问题 → 选牌阵 → 点击/摇一摇洗牌 → 扇形抽牌 → 逐张翻牌 → 查看概要解读 → 点击「深层解读」解锁完整解读 → 保存到记录。
