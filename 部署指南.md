代码已成功推送到你的 GitHub 仓库：
**https://github.com/Sanyueprince/Moon-mirror-tarot**

下面只剩最后一步——在网页上开启 GitHub Pages，就能得到可分享的公网链接。

---

## 一、开启 GitHub Pages（你在网页操作，约 1 分钟）

1. 打开仓库主页：https://github.com/Sanyueprince/Moon-mirror-tarot
2. 点击顶部的 **Settings**（设置）
3. 在左侧菜单找到并点击 **Pages**
4. 在 **Build and deployment → Source** 处，选择 **Deploy from a branch**
5. 在 **Branch** 处：
   - 分支选 **main**
   - 文件夹选 **/ (root)**
   - 点击 **Save**
6. 等待 1～2 分钟，刷新该页面，顶部会出现一行绿色提示：
   **"Your site is live at ..."**

## 二、你的公网链接

开启成功后，访问地址为：

```
https://sanyueprince.github.io/Moon-mirror-tarot/
```

把这个链接发给朋友，**任何人在手机或电脑浏览器打开即可使用**，无需和你同一 WiFi，也无需你的电脑开机。

> 提示：手机上打开后，可以「添加到主屏幕」，体验更接近 App。

## 四、常见问题

**Q：打开链接显示 404？**
- Pages 刚开启需要 1～2 分钟生效，稍等后刷新。
- 确认 Settings → Pages 里分支选的是 `main` + `/ (root)`。
- 注意链接大小写：仓库名是 `Moon-mirror-tarot`。

**Q：朋友打开后标题字体看起来普通？**
- 本项目标题使用思源宋体（在线加载），若访客网络无法访问 Google Fonts，会自动回退到系统宋体（苹方/宋体），不影响阅读与使用。

**Q：占卜记录会同步吗？**
- 不会。记录保存在每个人自己手机/浏览器的本地存储里，各自独立，互不影响。这是纯前端 Demo 的正常表现。

**Q：推送时报网络错误（HTTP2 framing / 连接超时）？**
- 多为网络波动，重试 `git push` 即可；本项目首次推送已成功。
